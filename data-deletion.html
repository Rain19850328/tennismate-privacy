데이터 삭제 요청 안내

테니스메이트 · TENNIS MATE

본 페이지는 테니스메이트(TENNIS MATE) 앱 이용자가 자신의 데이터를 삭제하는 방법을 안내합니다.

앱/운영자: 테니스메이트 · 운영자 한상훈

1. 앱에서 직접 삭제 (즉시)

대부분의 데이터는 앱 안에서 이용자가 직접, 즉시 삭제할 수 있습니다.

  • 회원 삭제: ‘회원’ 탭에서 해당 회원의 삭제 (사진·전화·프로필 등 개인정보 제거)
  • 경기·팀 삭제: ‘기록’ 탭에서 경기 삭제, ‘대진’에서 팀 삭제
  • 시즌 초기화 / 클럽 초기화: 경기 기록 또는 클럽 전체 데이터 삭제
  • 공유 클럽 나가기: 이 기기의 목록에서 제거

2. 공유(클라우드) 데이터 완전 삭제 요청

공유 클럽의 서버(클라우드) 데이터까지 완전히 삭제하려면 아래로 요청해 주세요.

  1. 아래 이메일로 삭제할 클럽의 참여코드(6자리)와 함께 삭제를 요청합니다.
  2. 본인 확인 후 해당 클럽의 서버 데이터를 삭제해 드립니다.

문의(삭제 요청) 이메일: hynbliss@gmail.com

3. 삭제되는 데이터 / 보관 기간

  • 삭제되는 데이터: 회원 정보(이름·사진·성별·전화번호·년생·구력·NTRP), 경기·순위·팀·토너먼트 기록, 클럽 정보 등 이용자가 입력한 모든 데이터.
  • 추가 보관 기간: 없음. 이용자가 삭제(또는 삭제 요청)하면 해당 데이터는 지체 없이 제거되며, 별도로 추가 보관하지 않습니다.
